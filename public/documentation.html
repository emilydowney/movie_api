Movie API

This web application will provide clients with information about different movies, directors, and genres. Users will also be able to sign up on the app, update their information, and form a list of their favorite movies.

Request URL HTTP Method Parameters Request Body Response Body
Retrieves a list of all available movies. /movies GET N/A None A JSON object holding data of all the movies available.

Example: 
      [
        {
            "name": "The Avengers",
            "director": "Joss Whedon"
        },
        {
            "name": "The Lord of the Rings: Return of the King",
            "director": "Peter Jackson"
        }
      ]
Retrieves data about a single movie by title. /movies/[Name] GET /[Name]

Example: /movies/The%20Avengers		 None A JSON object holding data of a single movie.

Example: 
      [
        {
            "name": "The Avengers",
            "director": "Joss Whedon"
        }
      ]
Retrieves data about a particular genre. /genres/[Name] GET /[Name]

Example: /genres/genreName		 None Text data about a certain genre based on genre name.

Example:
Thriller
Example text here.
Retrieves data about a certain director by name. /directors/[Name] GET /[Name]

Example: /directors/directorName		 None A JSON object holding data about a single director.

Example: 
      [
        {
            "name": "Joss Whedon",
            "birth year": "1964"
        }
      ]
Allow new users to register /users POST N/A A JSON object holding the new user's data.

Example: 
      {
       "username" : "newuser"
      }
A JSON object holding data about the new user added plus a unique ID.

Example: 
      {
        "username" : "newuser",
        "id" : "12345"
      }
Allow a user to update their username. /users/[Username] PUT /[Username]

Example: /users/emdowney		 None A message indicating the update was successful.
Allow users to add a movie to their favorites list. /users/[Username]/favorites POST /[Username]/favorites

Example: /users/emdowney/favorites		 None A message indicating the movie was added successfully to the user's favorites.
Allow users to delete a movie from their favorites. /users/[Username]/favorites DELETE /[Username]/favorites

Example: /users/emdowney/favorites		 None A message indicating the deletion was successful.
Allow existing users to deregister /users/[Username] DELETE /[Username]

Example: /users/emdowney		 None A message indicating the user deregistered successfully.
