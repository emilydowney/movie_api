Movie API

This web application will provide clients with information about different movies, directors, and genres. Users will also be able to sign up on the app, update their information, and form a list of their favorite movies.

Request URL HTTP Method Parameters Request Body Response Body
Retrieves a list of all available movies. /movies GET N/A None A JSON object holding data of all the movies available.

Example: 
      [
        {
            "name": "The Avengers",
            "director": "Joss Whedon"
        },
        {
            "name": "The Lord of the Rings: Return of the King",
            "director": "Peter Jackson"
        }
      ]
Retrieves data about a single movie by title. /movies/[Title] GET /[Title]

Example: /movies/The%20Avengers		 None A JSON object holding data of a single movie.

Example: 
      [
        {
            "name": "The Avengers",
            "director": "Joss Whedon"
        }
      ]
Retrieves data about a particular genre. /genres/[Name] GET /[Name]

Example: /genres/Fantasy		 None A JSON object holding data about a certain genre based on genre name.

Example:
        [
          {
              "name": "Fantasy",
              "description": "Example description here."
          }
        ]
Retrieves data about a certain director by name. /directors/[Name] GET /[Name]

Example: /directors/Joss%20Whedon		 None A JSON object holding data about a single director.

Example: 
      [
        {
            "name": "Joss Whedon",
            "birth year": "1964"
        }
      ]
Allow new users to register /users POST N/A A JSON object holding the new user's data.

Example: 
      {
       "username" : "newuser"
      }
A JSON object holding data about the new user added plus a unique ID.

Example: 
      {
        "username" : "newuser",
        "id" : "12345"
      }
Allow a user to update their information. /users/[Username] PUT /[Username]

Example: /users/user1		 A JSON object holding the data to be updated.

Example: 
        {
          "username" : "newuser",
          "password" : "12345"
        }
A JSON object holding the updated data.

Example: 
        {
          "username" : "newuser",
          "password" : "12345"
        }
Allow users to add a movie to their favorites list. /users/[Username]/favorites/[MovieID] POST /[Username]/favorites[MovieID]

Example: /users/user1/favorites/123		 None A JSON object holding the user's updated favorites list.

Example: 
        {
          "username" : "user1",
          "favorites" : "123"
        }
Allow users to delete a movie from their favorites. /users/[Username]/favorites/[MovieID] DELETE /[Username]/favorites/[MovieID]

Example: /users/user1/favorites/123		 None A JSON object holding the user's updated favorites list.

Example: 
        {
          "username" : "user1",
          "favorites" : []
        }
Allow existing users to deregister /users/[Username] DELETE /[Username]

Example: /users/user1		 None A message indicating the user deregistered successfully.

Example:
user1 was deleted.
